### System requirements
Android 9.0+ (API level 28+)

### Differences from [LAMPA](https://github.com/lampa-app/LAMPA)
- increase the minimum system requirement to Android 9.0 (API 28)
- removal of the legacy Crosswalk engine
- obsolete code for older Android versions
- update core dependencies and android sdk
